Web Starters

Get yourself a running start

Some startkits for building quick prototypes or websites. Depending on what you want to achieve, a different start is provided and ready to go.

Find me on Github

Startkits

Small

The Small Web Starter includes all you need to start a simple prototype or a very small website. A barebone index.html is provided, Gulp will automate all your frontend-tasks, Scss will be used to write your css and Bower will manage your vendors.

Find me on Github

Medium

The Medium Web Starter includes all you need to start a medium scaled website. A barebone index.html is provided, Gulp will automate all your frontend-tasks, Scss will be used to write your css, Bundler will manage your ruby-gems, Bower will manage your vendors and hologram will provide your styleguide.
